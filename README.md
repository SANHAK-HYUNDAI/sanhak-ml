# sanhak-ml
## Purpose
- Machine Learning을 활용한 데이터 정제 및 유사도 분석

## Performance
- Upload
  - 데이터 정제
  - 클러스터링, 자체 유사도 분석을 통한 고장/불량 관련 데이터 추출
  - 자동차 수리 정보 RO데이터와의 유사도 분석
- Category
  - bigcategory 빈도 수 분석
  - 상위의 대 카테고리에 따른 subcategory 빈도 수 분석
- Wordcloud
  - Counter를 활용한 단어 빈도수 분석
  - 명사 형태소 분석 시 데이터 정제 필요
